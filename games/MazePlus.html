MazePlus

Description: User navigates maze on cube faces and seeks target
Programmer: Matthew
Compiler: Smart Mobile Studio 2.2

Introduction

This web version of MazePlus is designed as a preview to be run in a browser, preferably on a PC. You can use the wasd or arrow keys or the buttons to move to the destination 'O'. If the game does not work in your current browser, please try another such as Chrome. On a Raspberry Pi, the display might look better if you install the Lucida Console font.

If the program does not work after clicking on the display, try another browser such as Chrome.

Technical Features

The program benefits from:

  • own procedures;
  • use of inbuilt routines such as Randomize, RandomInt, FillRectF, FillTextFLineToF, MoveToF, GetImageData and BeginPath;
  • appropriate use of case statements;
  • handling of keyboard input;
  • enumerated type.
