TabBar sencha-like

Description: SMS clone Sencha Touch TabBar visual component.
Programmer: Lynkfs
Compiler: Smart Mobile Studio 2.2

Introduction

One thing that I don't like in Smart Mobile Studio and tend to disagree is the RAD approach to build, develop, and manage RIA (rich internet applications). I sincerely do not like of the RAD methodology. I prefer hand-written code, I consider to be more readable, more maintainable and believe to offer more customizations and abstractions. I'm just curious and styled a TabBar visual component created by Lynks.

Developed with SMS Enterprise 2.2.0.4093 and tested/working with 2.1
Copy the tabBar.pas file to SMS_INSTALL_DIR/Libraries
Copy the tabBar.spk file to SMS_INSTALL_DIR/Packages
Copy the TtabBar.png file to SMS_INSTALL_DIR/Packages/Glyphs
Start SMS and see at "3rdParty" tab, we have a new component: tabBar.
Open the project "TesttabBar.sproj", at project Options "Use custom theme", there're some Sencha Touch Styles you have to apply to the project.
