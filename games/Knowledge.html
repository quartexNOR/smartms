Knowledge

Description: Keep the taxi on the road
Programmer: Peter Hearnshaw
Compiler: Smart Mobile Studio 2.2

Version I

This program demonstrates:
  • the display of a screen of instructions while loading images;
  • the conversion of the canvas to ImageData to access the getPixel function;
  • the display of images with Canvas.DrawImageF, lines with Canvas.LineF, rectangles with Canvas.FillRectF, circles with Canvas.Ellipse, and a polygon for the arrow head by creating a path then filling it;
  • input from the mouse and keyboard.

Version II

This version of Peter Hearnshaw's Knowledge will run in Chrome, Firefox, Opera, within Smart Mobile Studio and in some versions of Internet Explorer. The conversion of this program from Peter Hearnshaw's original code (that used Pascal with WinGraph) demonstrates:

  • the display of a screen of instructions while loading images;
  • the display of images with Canvas.DrawImageF, lines with Canvas.LineF, rectangles with Canvas.FillRectF, circles with Canvas.Ellipse, and a polygon for the arrow head by creating a path then filling it;
  • input from the mouse and keyboard.
