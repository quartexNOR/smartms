RoamingBlockBuster

Description: User busts the randomly moving blocks by firing at them
Programmer: Joel Sutcliffe
Compiler: Smart Mobile Studio 2.2

Introduction

Joel shows how you can write a shooter game in Smart Mobile Studio with a moderate amount of code. Use the wasd or arrow keys to move and q or the space bar to fire (in the direction of your last movement) at the pink targets. The targets move randomly in a mesmerising manner and are not easy to hit. In this version of the game you can see how many blocks you can bust until, say, the time reaches 50. You may like to introduce your own scoring system and make the game either easier or more difficult to play. The code follows the game in action. If RoamingBlockBuster does not run in your current browser, please try another (such as Chrome).

Features

Object-oriented code, keyboard input, RandomInt, FillRect, FillRectF
